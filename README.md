📌 About This Project
A machine learning project that predicts the Order Status of e-commerce orders such as Delivered, Shipped, Cancelled, Returned, and Pending. Built using Decision Tree Classifier and Random Forest Classifier trained on a real dataset.

🎯 What It Does
Loads historical regional transaction records.
Encodes categorical data for tracking.
Analyzes order costs and quantities.
Splits data into evaluation sets.
Trains a predictive Decision Tree.
Builds a robust Random Forest.
Evaluates accuracy using confusion matrices.
Forecasts returns for new orders.
🛠️ Skills Used
Python and Pandas for data handling
Scikit-learn for machine learning
Matplotlib and Seaborn for charts
Supervised learning and classification
Model comparison and evaluation techniques
🤖 Models Used
Decision Tree Classifier — simple and visual flowchart based model
Random Forest Classifier — stronger model that combines multiple decision trees

🚀 How to Run
Open the project folder in VS Code.
Ensure you have your virtual environment (venv) activated in your terminal.
Open the file Project2_Data_Classification.ipynb.
Select your Python/venv interpreter kernel in the top-right corner of VS Code.
Make sure your code cell language is set to Python (not Plain Text).
Run each cell one by one using Shift + Enter or by clicking the Play icon.
